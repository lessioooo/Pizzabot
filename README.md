# Pizzabot

Pizzabot is a high performance, multi functional Discord Bot designed mostly for fun by me, made in JavaScript with the help of Node.js.

## Key features

* **Dice Rolling:** A dynamic parser built to evaluate complex dice notations (e.g., `1d20`, `2d6`) for tabletop gaming utilities.
* **Asynchronous Architecture:** Event-driven design ensuring low-latency handling of gateway events and API requests.
* **Real-Time Weather Data:** Integration with a meteorological API to fetch and display current weather conditions and core climate metrics for any global city.

## Technologies used
* **Runtime:** Node.js
* **Language:** JavaScript
* **Framework:** Discord.js
* **Data Integration:** REST API for weather fetching

## Getting Started

To get a local development instance of Pizzabot up and running, follow these steps.

### Prerequisites

Ensure you have the following installed and configured on your local machine:
* [Node.js](https://nodejs.org/) (v16.x or higher recommended)
* [npm](https://www.npmjs.com/)
* A registered bot application via the [Discord Developer Portal](https://discord.com/developers/applications)

### Installation

1. Clone the repository:
```bash
   git clone [https://github.com/lessioooo/Pizzabot.git](https://github.com/lessioooo/Pizzabot.git).
```
2. Navigate into the project directory:
```bash
  cd Pizzabot
```
3. Install the required dependencies:
```bash
  npm install
```
 ### Configuration
 Create a .env file in the root directory to securely store your environment variables and API keys. Do not commit this file to version control.
 ###Running the Bot
 To start the bot, run the following command in your terminal:
 ```bash
  node index.js
 ```
